# Currency Exchange Rate App

An Android application that displays real-time currency exchange rates from GBP to various world currencies.

## Features

- Real-time currency exchange rates from RSS feed
- Currency conversion calculator
- Flag icons for each currency
- Search and filter currencies
- Auto-refresh every hour
- Offline support with cached data
- Color-coded rates for easy visualization

## Technical Stack

- Java
- Android SDK
- XML RSS parsing
- LiveData & ViewModel (MVVM architecture)
- RecyclerView for efficient list display
- Material Design UI components

## Getting Started

1. Clone this repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application on an emulator or device

## Requirements

- Android Studio Arctic Fox or later
- Android SDK API 21 (Lollipop) or higher
- Internet connection for fetching exchange rates
